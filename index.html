3 IN 1: RESTORE ANTIQUES

MANAGE CATALOG OF ALL ITEMS

BE AWARE OF WORK CLIMATE

If you work in a restoration workshop, evaluate our system for keeping records on the restoration processes performed and controlling climate during the restoration process. It will help you organize and control the restoration process from beginning to end.

  1. 1. Create users accounts

    There are two roles in a system - User and Super User. The second one manage the whole system, create accounts for the team and check all restoration projects.

  2. 2. Activate sensors

    Everybody can have a sensor which provides Temperature and Humidity measures every 2 sec. Use a sensor while you start working on a new object. Data from the sensor is logged in a project history.

  3. 3. Use checklists

    While working tick steps done. Make checklists by yourself or edit existing ones. Describe every step of the restoration process including materials used and time of each step of work.

  4. 4. Make catalog of restorated object

    Every project takes a part in a catalog. Specialists can look at all projects of the team and show them to clients.
